## Find working Daml code by topic

<Callout>**Problem.** Forum threads contain working code but Google rarely surfaces them.</Callout>

**Solution.** `find_code_examples(topic)` returns forum threads with embedded `&lt;pre&gt;`/`&lt;code&gt;` blocks matching the topic. Faster than reading the spec when you need a sketch.
